```jsx file=fancy-button.jsx
// here's a button in React!
<button
  onClick={() => {
    alert('Hello MDX!');
  }}

```

Wow! Such code snippets!
Let's see another, with line highlighting:

```js highlights=1,4-6
// fizzbuzz in JS
for (let i = 1; i <= 100; i++) {
  let out = '';
  if (i % 3 === 0) out += 'Fizz';
  if (i % 5 === 0) out += 'Buzz';
  console.log(out || i);
}
```
